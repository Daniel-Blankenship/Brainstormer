Project Movie Match Requirements

Team members

Abbott and Costello

Purpose

This project allows members to rent videos to stream online.

Requirements

  1. Name: add_member
    Type: functional
    Summary: Members should be added to the database. New members contain a variety of information that is used to manage rental accounts.

  2. Name: add_movies
    Type: functional
    Summary: Movies addition functionality should be added.

  3. Name: delete_movies
    Type: functional
    Summary: Certain movies may be removed from the movie list if necessary.

  4. Name: platform_os
    Type: constraint
    Summary: The target platform operating system is Windows 10.

  5. Name: platform_resolution
    Type: constraint
    Summary: Must work on 4K monitors.

  6. Name: checkout_payment_secure
    Type: risk
    Summary: Payment information shall be transmitted in a secure fashion.
